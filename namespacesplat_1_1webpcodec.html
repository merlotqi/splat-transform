std::tuple<std::vector<uint8_t>, int, int> splat::webpcodec::decodeRGBA ( const std::vector< uint8_t > &  webp)

Decodes a WebP image to RGBA format.

This function decodes a WebP-encoded image into raw RGBA pixel data. It uses the WebP library's RGBA decoder to convert the compressed WebP data into an uncompressed RGBA byte array.

Parameters
webpInput WebP-encoded image data as a vector of bytes
Returns
std::tuple containing:
  • std::vector<uint8_t>: Decoded RGBA pixel data, arranged as [R0, G0, B0, A0, R1, G1, B1, A1, ...] for each pixel row-major
  • int: Width of the decoded image in pixels
  • int: Height of the decoded image in pixels

Encodes RGBA image data to lossless WebP format.

This function compresses raw RGBA pixel data into a lossless WebP image. It uses WebP's lossless encoding mode which preserves exact pixel values while providing good compression for RGBA images.

Parameters
rgbaInput RGBA pixel data as a vector of bytes, arranged as [R0, G0, B0, A0, R1, G1, B1, A1, ...] for each pixel row-major
widthWidth of the input image in pixels
heightHeight of the input image in pixels
strideNumber of bytes between consecutive rows. If 0, stride is calculated as width * 4 (standard RGBA layout)
Returns
std::vector<uint8_t> Lossless WebP-encoded image data
