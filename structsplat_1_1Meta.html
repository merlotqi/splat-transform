Detailed Description

Metadata structure for Gaussian Splatting data representation.

The Meta struct contains comprehensive metadata for Gaussian Splatting data, including version information, asset generation details, and file organization for various data components (means, scales, rotations, spherical harmonics). It supports parsing from and encoding to JSON format.

Encode metadata to JSON string.

Returns
JSON string representation of the metadata

Converts all metadata fields to a JSON object following the same structure expected by parseFromJson. The output is compact (no whitespace) for efficiency.
