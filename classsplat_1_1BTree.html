Detailed Description

Bounding Volume Hierarchy (BVH) tree implementation using Axis-Aligned Bounding Boxes (AABBs).

This class constructs a binary bounding volume hierarchy tree for spatial partitioning of centroid data. The tree accelerates spatial queries by hierarchically grouping data points into nested bounding boxes.
